# Xcarus茶思屋 - 内容发布 SOP

> 从写文章到上线，3 步搞定。全程不需要手写 Hugo 头文件。

---

## 🚀 快速开始（写完文章到上线）

### 1. 写文章

打开任意 Markdown 编辑器，正常写作：

```markdown
# 带电粒子在磁场中的运动方向判断

这篇文章聊聊洛伦兹力的判断方法。

## 原理

带电粒子在匀强磁场中...

## 解题技巧

左手定则记住一句话：磁力线穿手心，四指指向正电荷运动方向...
```

**不需要写 `---` 头文件**，工具会自动帮你生成。

### 2. 发布

把 `.md` 文件**拖进 `_drafts/` 目录**，然后运行：

```bash
python publish.py
```

工具会：
- 自动提取标题
- 自动归类
- 自动生成头文件
- 创建 Hugo 文章目录

### 3. 推送上线

```bash
cd xcarus-blog
git add .
git commit -m "新文章: XXXX"
git push origin main
```

等 2 分钟，访问 https://alexerrorx.github.io/ 即可看到。

## 📂 目录结构

```
xcarus-blog/
├── _drafts/              ← 📝 把你写的 .md 放这里
│   └── 我的新文章.md
├── content/
│   └── posts/            ← 📤 工具自动生成的文章目录
│       └── 我的新文章/
│           └── index.md
├── publish.py            ← 🔧 一键发布脚本
└── PUBLISH.md            ← 📖 本文档
```

---

## 🔧 工具功能

`publish.py` 自动完成：

| 步骤 | 说明 |
|------|------|
| 提取标题 | 从第一个 `# 标题` 自动提取 |
| 归类 | 根据文章内容自动匹配分类 |
| 生成标签 | 自动提取关键词作为标签 |
| 生成摘要 | 截取第一段作为文章简介 |
| 创建目录 | 生成 Hugo Page Bundle（文件夹 + index.md） |

## 📝 文章格式参考

### 最简写法（推荐）

```markdown
# 文章标题

第一段内容...

## 二级标题

- 列表
- 列表

**加粗** *斜体*
```

### 带图片

把图片和 .md 文件放在一起，然后 `_drafts/` 目录结构：

```
_drafts/
├── 我的文章.md
├── 示意图.jpg
└── 另一张图.png
```

文章中引用：
```markdown
![示意图](./示意图.jpg)
```

工具发布时，这些图片会一起被复制到文章目录。

## ❓ FAQ

**Q: 工具把我的文章归错类了怎么办？**
A: 在 `.md` 文件开头加一行注释：
```
<!-- category: 物理与科学 -->
```
工具会优先使用你指定的分类。

**Q: 怎么修改端口？**
A: `hugo server -D -p 你想要的端口号`

**Q: 忘记 git push 了？**
A: 文章已经在 `content/posts/` 里了，随时可以 `git push`
